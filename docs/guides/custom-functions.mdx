## Custom Functions

Browserable provides a flexible way to extend its functionality through custom tools and functions. This allows you to add specialized capabilities to your browser automation tasks.

### Current Status

We are actively developing an easy-to-use framework for adding custom tools to Browserable. This will include:

- Simple API for adding custom functions
- Built-in templates for common scenarios:
  - User input handling
  - Form management
  - Authentication flows
  - And more...
