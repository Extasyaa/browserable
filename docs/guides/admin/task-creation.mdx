## Task Creation Interface

The task creation interface provides a visual way to create and manage browser automation tasks.

<img class="border-2" src="/images/browserable-admin-create-task.png" />

### Current Access

You can access the task creation interface at:
```
http://localhost:2001/dash/@admin/new-task
```
